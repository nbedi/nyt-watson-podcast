# nyt-watson-podcast

Here's IBM's Watson with the New York Times morning briefing

Site at http://neilbedi.com/nyt-watson-podcast

## Installation

1. Clone repo
2. Set up IBM Bluemix account
3. Save IBM Watson API auth params in watson-params.json
4. Run node index.js
